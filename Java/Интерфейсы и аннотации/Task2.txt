Интерфейс, объявленный как public interface Interface { ... }, имеет публичную область видимости. 
Это означает, что интерфейс доступен для использования в любом другом классе, находящемся в любом пакете, при условии, что в нем указан правильный импорт. 
Модификатор public позволяет этому интерфейсу быть доступным по всему проекту, независимо от пакета, в котором он был объявлен.
