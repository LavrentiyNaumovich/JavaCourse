Ключевое слово this используется для обращения к текущему объекту класса, в котором оно применяется. 
В подклассе может использоваться для вызова методов или доступа к полям текущего объекта, а также для вызова конструктора текущего класса.
Ключевое слово super используется для обращения к родительскому классу.
В подклассе оно применяется для вызова конструктора родительского класса, доступа к методам, полям родителя и для переопределения методов родительского класса.
