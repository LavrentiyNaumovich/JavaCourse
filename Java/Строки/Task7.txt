Метод flush() в Java используется для принудительной записи всех данных из буфера в соответствующий поток
Обычно данные сначала накапливаются в буфере (например, при записи в файл или выводе на экран), чтобы уменьшить количество операций ввода-вывода 
Метод flush() заставляет сразу записать все эти данные, а не ждать, когда буфер будет полностью заполнен или поток будет закрыт
Это полезно, когда нужно, чтобы данные были немедленно доступны, например, при выводе прогресса или ошибок в процессе выполнения программы
