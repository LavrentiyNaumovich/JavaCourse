Разница между циклом(1) и вызовами (2):
Цикл for (1): При вызове метода printNum(n) с элементами массива типа Number Java будет выбирать наиболее подходящий метод в зависимости от конкретного типа каждого элемента. 
Например:
Если элемент массива n имеет тип Integer, будет вызван метод printNum(Number n).
Если элемент массива n имеет тип Float, будет вызван метод printNum(Float f).

Вызовы (2): В этих вызовах Java напрямую использует типы, передаваемые в метод, и вызывает конкретные перегруженные методы:
Для примитивов типа int, float, и double Java вызывает перегруженные методы с соответствующими типами.
Для объектов типа Integer будет вызван метод, который принимает Integer, а для объектов типа Float — метод, принимающий Float.

В цикле Java вызывает метод для каждого типа данных, исходя из их реального типа (например, Integer, Float и т.д.), а в вызовах
явно использует примитивы и объекты, что приводит к вызову конкретных перегруженных методов, учитывая точный тип.
