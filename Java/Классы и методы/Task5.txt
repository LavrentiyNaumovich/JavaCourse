static logic
logic (1) id= 0
logic (2) id= 1
ctor id= 100

static logic: Выводится первым, потому что статический блок выполняется один раз при первой загрузке класса.
logic (1) id= 0: Выводится следующим, потому что объект ещё не полностью инициализирован, 
и на момент выполнения этого блока поле id имеет значение по умолчанию — 0.
logic (2) id= 1: Выводится после того, как поле id было инициализировано значением 1.
ctor id= 100: Выводится в самом конце, когда конструктор присваивает новое значение полю id — 100.
